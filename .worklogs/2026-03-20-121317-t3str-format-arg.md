# Move --format flag to subcommands

## Summary

Moved `--format` from the top-level CLI struct to each subcommand (extract, run) so it can appear after the subcommand name. The pipeline invokes `t3str extract --language ... --format json` which requires --format to be a subcommand arg, not a global arg.

## Key Files

- `apps/t3str/crates/adapters/inbound/cli/src/main.rs` — format arg moved to each Command variant
