# Ruby: broader exception rescue + compat error detection

## Summary
Fixed minitest direct runner to rescue all exceptions (not just LoadError/NameError) and added compatibility error detection for Rails/ActiveSupport dependency failures.

## Changes
- `rescue Exception` instead of `rescue LoadError, NameError` — catches Bundler::GemNotFound and other deep errors
- Exclude `integration/` test files (Rails integration tests can't run standalone)
- Added `detect_ruby_compat_error()` that reports Bundler gem conflicts and missing ActiveSupport/Rails as `TestStatus::Error` instead of silent 0
