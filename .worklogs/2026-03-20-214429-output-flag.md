# t3str v0.1.9: --output flag for file-based results

## Summary
Added `--output <path>` flag to `t3str run` that writes full JSON to a file instead of stdout. Stdout gets a compact summary. Fixes DomainDetective's 632KB output being truncated by the shedul3r SDK.
