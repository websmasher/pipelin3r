# Steps 7-8: Run parsers and classify results

**Date:** 2026-03-19 12:04
**Task:** Create s07_run_parsers.rs and s08_classify.rs in the dev-process-v3 steps module.

## Goal
Two synchronous pipeline steps that process extracted test data into ground-truth fixtures and classify cross-library agreement.

## Approach

### Step 7 — s07_run_parsers.rs
1. Discover `research/{lang_slug}/*-tests.json` files across all languages
2. Parse each as serde_json::Value, extract test inputs from known keys
3. Dedup by content, assign deterministic fixture IDs (sorted counter-based)
4. Write `ground-truth/{fixture_id}/input.txt` and per-source JSON files
5. Write `ground-truth/manifest.json`

### Step 8 — s08_classify.rs
1. Read manifest, iterate fixtures
2. Read all `{lang}-{lib}.json` files per fixture
3. Compare normalized JSON values for agreement classification
4. Write per-fixture `classification.json` and global `summary.json`
