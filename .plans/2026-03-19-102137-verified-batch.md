## Approach

### 1. Add `run_verified_step_batch` to pipelin3r

In `verified/orchestrator.rs`, add:
```rust
pub async fn run_verified_step_batch<T, F>(
    executor: &Executor,
    work_dir: &Path,
    items: Vec<T>,
    concurrency: usize,
    mapper: F,
) -> Vec<(T, Result<VerifiedStepResult, PipelineError>)>
where
    T: Send + 'static,
    F: Fn(&T) -> VerifiedStep + Send + Sync + Clone + 'static,
```

Uses `run_pool_map` internally. Each item gets its own `run_verified_step` call.
