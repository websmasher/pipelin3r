# Install all language runtimes in the claude-worker image

**Date:** 2026-03-19 15:34
**Task:** Make the worker container self-sufficient with all runtimes baked in

## Approach
Two changes to /Users/tartakovsky/Projects/websmasher/claude-worker:

1. `railpack.json` — add apt packages for runtimes available via apt
2. `scripts/install-deps.sh` — download standalone runtimes for those not in apt or needing newer versions

## What to install

| Runtime | Method | Why |
|---------|--------|-----|
| Python 3.12 | apt `python3 python3-pip python3-venv` | System python is fine |
| Go 1.22+ | standalone tarball | apt version may be too old |
| Rust | rustup in install-deps.sh | Not in apt |
| PHP 8.3+ | apt `php-cli` or standalone | Need 8.3+ for composer deps |
| Composer | download phar in install-deps.sh | Not in apt |
| Ruby 3.x | apt `ruby ruby-bundler` | System ruby is fine |
| .NET 10 | dotnet-install.sh in install-deps.sh | Not in apt |
| Node 22 | already in railpack.json | Already there |
| gcc/make | apt `build-essential` | Needed for Rust native deps |
| git/curl | already in railpack.json | Already there |

## Files to modify
- `claude-worker/railpack.json` — add aptPackages
- `claude-worker/scripts/install-deps.sh` — add runtime downloads
