# Fix 5 bugs in pipelin3r agent module (bugs 10-14)

**Date:** 2026-03-17 15:45
**Task:** Fix security and correctness bugs in agent/mod.rs and agent/execute.rs

## Goal
Fix 5 bugs: path traversal in expect_outputs, missing work_dir validation, symlink escape in upload, silent error swallowing, and shared work_dir in batch.

## Approach

### Bug 10: expect_outputs path traversal
- Make `validate_path` in bundle/mod.rs `pub(crate)`
- Call it in execute_with_work_dir before downloading each expected output

### Bug 11: No work_dir path validation
- Add `validate_work_dir(path: &Path) -> Result<(), PipelineError>` in execute.rs
- Call at start of AgentBuilder::execute() and execute_single_task()

### Bug 12: Symlink escape in remote upload
- In read_dir_to_memory and collect_relative_paths, canonicalize base path once
- Pass canonical base into inner functions, check each entry stays within base

### Bug 13: Silent error swallowing
- Change `.ok()` to `?` propagation in execute_dry_run_capture line 65-67

### Bug 14: Shared work_dir in batch
- In AgentBatchBuilder::execute(), collect work_dir paths, check for duplicates using BTreeSet

- `packages/pipelin3r/src/agent/execute.rs` — bugs 10, 11, 12, 13
