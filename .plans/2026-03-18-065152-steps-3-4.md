# Implement steps 3-4 in dev-process-v2

**Date:** 2026-03-18 06:51
**Task:** Add steps 3 (extract structured libraries) and 4 (extract tests) to dev-process-v2.

## What each step does

### Step 3: Extract structured libraries from research
- Input: `research/{lang}/overview.md` (from step 2)
- Output: `research/{lang}/libraries.json` — structured list matching v1 format:
  ```json
  [{"name": "eikendev/sectxt", "url": "https://github.com/eikendev/sectxt", "description": "...", "language": "rust", "test_dir": ".", "clone_note": "..."}]
  ```
- This is an LLM step: the agent reads the overview.md and extracts structured data
- Runs per language with concurrency

### Step 4: Extract tests from libraries
- Input: `research/{lang}/libraries.json` (from step 3)
- Output: `research/{lang}/{library-slug}-tests.json` — extracted test cases from each library
- The agent clones the repo (or reads it via web), finds test files, extracts test functions/fixtures
- Runs per library with concurrency
