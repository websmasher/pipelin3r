# t3str CLI Foundation Setup

**Date:** 2026-03-20 09:38
**Task:** Create directory structure, workspace Cargo.toml, config files, and domain/types crate for t3str CLI app.

## Goal
Working t3str workspace with domain/types crate that compiles under strict clippy/lint rules matching shedul3r patterns.

## Approach
1. Create directory structure for all 6 crates
2. Create workspace Cargo.toml modeled on shedul3r
3. Create config files (rust-toolchain.toml, rustfmt.toml, deny.toml, clippy.toml)
4. Create domain/types crate with all source files
5. Verify compilation with `cargo build`

## Files to Create
- `apps/t3str/Cargo.toml` — workspace root
- `apps/t3str/rust-toolchain.toml` — toolchain pinning
- `apps/t3str/rustfmt.toml` — formatting config
- `apps/t3str/deny.toml` — dependency audit
- `apps/t3str/clippy.toml` — workspace-level clippy config
- `apps/t3str/crates/domain/types/Cargo.toml` — domain types crate
- `apps/t3str/crates/domain/types/clippy.toml` — domain layer clippy bans
- `apps/t3str/crates/domain/types/src/*.rs` — domain type source files
- Stub directories for remaining 5 crates
